еҲӣе»әе’Ңз®ЎзҗҶSpec

еӯҰд№ дҪҝз”Ё HarnSpec Specзҡ„еҹәжң¬ж“ҚдҪңпјҡеҲӣе»әгҖҒжӣҙж–°гҖҒжҹҘзңӢе’Ңз®ЎзҗҶSpecз”ҹе‘Ҫе‘ЁжңҹгҖӮ

еҲӣе»әж–°SpecвҖӢ

еҹәжң¬еҲӣе»әвҖӢ

harnspec create my-feature

иҝҷдјҡеҲӣе»әдёҖдёӘж–°SpecпјҢеҢ…еҗ«пјҡ

  • иҮӘеҠЁз”ҹжҲҗзҡ„е”ҜдёҖ IDпјҲдҫӢеҰӮпјҢ042-my-featureпјү
  • еҲқе§ӢеҢ–зҡ„frontmatterпјҲзҠ¶жҖҒпјҡplannedпјҢеҲӣе»әж—Ҙжңҹпјү
  • specs/042-my-feature/README.md дёӯзҡ„еҹәжң¬SpecжЁЎжқҝ

дәӨдә’ејҸеҲӣе»әвҖӢ

harnspec create

дёҚеёҰеҗҚз§°ж—¶пјҢжӮЁе°Ҷиў«дәӨдә’ејҸжҸҗзӨәпјҡ

  • SpecеҗҚз§°
  • еҸҜйҖүжҸҸиҝ°
  • еҲқе§ӢзҠ¶жҖҒпјҲй»ҳи®Өдёә plannedпјү

й«ҳзә§йҖүйЎ№вҖӢ

# дҪҝз”ЁеҲқе§ӢзҠ¶жҖҒеҲӣе»ә
harnspec create my-feature --status in-progress

# дҪҝз”Ёж ҮзӯҫеҲӣе»ә
harnspec create my-feature --tags api,backend

# дҪҝз”Ёдјҳе…Ҳзә§еҲӣе»ә
harnspec create my-feature --priority high

еҸҰи§ҒпјҡCLI еҸӮиҖғпјҡcreate иҺ·еҸ–жүҖжңүйҖүйЎ№

жӣҙж–°SpecвҖӢ

зҠ¶жҖҒжӣҙж–°вҖӢ

жӣҙж–°SpecзҠ¶жҖҒпјҲеҝ…йңҖ - ж°ёиҝңдёҚиҰҒжүӢеҠЁзј–иҫ‘frontmatterпјүпјҡ

# з§»иҮіиҝӣиЎҢдёӯ
harnspec update 042 --status in-progress

# ж Үи®°дёәе®ҢжҲҗ
harnspec update 042 --status complete

# е®ҢжҲҗеҗҺеҪ’жЎЈ
harnspec update 042 --status archived

еҸҜз”ЁзҠ¶жҖҒпјҡplannedгҖҒin-progressгҖҒcompleteгҖҒarchived

дјҳе…Ҳзә§е’Ңж ҮзӯҫвҖӢ

# и®ҫзҪ®дјҳе…Ҳзә§
harnspec update 042 --priority high

# ж·»еҠ /жӣҙж–°ж Үзӯҫ
harnspec update 042 --tags api,backend,refactor

# еҲҶй…Қз»ҷжҹҗдәә
harnspec update 042 --assignee alice

йҮҚиҰҒпјҡе§Ӣз»ҲдҪҝз”Ё harnspec update жӣҙж–°зі»з»ҹз®ЎзҗҶзҡ„еӯ—ж®өгҖӮжүӢеҠЁзј–иҫ‘дјҡжҚҹеқҸе…ғж•°жҚ®гҖӮ

еҸҰи§ҒпјҡCLI еҸӮиҖғпјҡupdate

жҹҘзңӢSpecвҖӢ

еҝ«йҖҹжҹҘзңӢвҖӢ

# жҹҘзңӢSpecпјҲж јејҸеҢ–д»Ҙдҫҝйҳ…иҜ»пјү
harnspec view 042

# жҹҘзңӢеӯҗ Specж–Үд»¶
harnspec view 042/DESIGN.md

еҺҹе§Ӣиҫ“еҮәвҖӢ

# иҺ·еҸ–еҺҹе§Ӣ markdownпјҲз”ЁдәҺи§Јжһҗпјү
harnspec view 042 --raw

# иҺ·еҸ–з»“жһ„еҢ– JSON
harnspec view 042 --json

еңЁзј–иҫ‘еҷЁдёӯжү“ејҖвҖӢ

# еңЁй»ҳи®Өзј–иҫ‘еҷЁдёӯжү“ејҖ
harnspec open 042

# жү“ејҖзү№е®ҡеӯҗ Spec
harnspec open 042/TESTING.md

еҸҰи§ҒпјҡCLI еҸӮиҖғпјҡview

еҲ—еҮәSpecвҖӢ

жҹҘзңӢжүҖжңүSpecвҖӢ

# еҲ—еҮәжүҖжңүSpec
harnspec list

# жҢүзҠ¶жҖҒеҲ—еҮә
harnspec list --status in-progress

# жҢүж ҮзӯҫеҲ—еҮә
harnspec list --tag api

зңӢжқҝи§ҶеӣҫвҖӢ

# еҸҜи§ҶеҢ–зңӢжқҝи§Ҷеӣҫ
harnspec board

жҳҫзӨәжҢүзҠ¶жҖҒеҲҶз»„зҡ„SpecеҸҠйЎ№зӣ®еҒҘеә·жҢҮж ҮгҖӮ

еҸҰи§ҒпјҡзңӢжқҝе’Ңз»ҹи®Ў

з®ЎзҗҶSpecж–Үд»¶вҖӢ

еӯҗ Specж–Үд»¶вҖӢ

еҜ№дәҺеӨҚжқӮSpecпјҢжӢҶеҲҶдёәеӨҡдёӘж–Үд»¶пјҡ

# еҲ—еҮәSpecдёӯзҡ„жүҖжңүж–Үд»¶
harnspec files 042

еёёи§Ғеӯҗ Specж–Үд»¶пјҡ

  • README.md - дё»SpecпјҲеҝ…йңҖпјү
  • DESIGN.md - иҜҰз»Ҷи®ҫи®Ў
  • IMPLEMENTATION.md - е®һзҺ°и®ЎеҲ’
  • TESTING.md - жөӢиҜ•зӯ–з•Ҙ
  • CONFIGURATION.md - й…ҚзҪ®зӨәдҫӢ

**дёәд»Җд№Ҳпјҹ**дҝқжҢҒжҜҸдёӘж–Үд»¶еңЁ 400 иЎҢд»ҘдёӢпјҲдёҠдёӢж–Үз»ҸжөҺ (Context Economy)еҺҹеҲҷпјүгҖӮ

еҸҰи§Ғпјҡеӯҗ Specж–Үд»¶

жҹҘзңӢж–Үд»¶еҲ—иЎЁвҖӢ

# жҳҫзӨәSpecдёӯзҡ„жүҖжңүж–Үд»¶
harnspec files 042

иҫ“еҮәпјҡ

рҹ“„ 042-my-feature дёӯзҡ„ж–Үд»¶

еҝ…йңҖпјҡ
  вң“ README.md        (234 иЎҢ)  дё»Spec

ж–ҮжЎЈпјҡ
  вң“ DESIGN.md        (187 иЎҢ)
  вң“ TESTING.md       (145 иЎҢ)

жҖ»и®Ўпјҡ3 дёӘж–Үд»¶пјҢ566 иЎҢ

еҪ’жЎЈSpecвҖӢ

еҪ“е·ҘдҪңе®ҢжҲҗе№¶дё”жӮЁжғіе°ҶSpecз§»еҮәжҙ»еҠЁи§Ҷеӣҫж—¶пјҡ

# еҪ’жЎЈе·Іе®ҢжҲҗзҡ„Spec
harnspec update 042 --status archived

еҪ’жЎЈзҡ„Specпјҡ

  • дёҚдјҡеҮәзҺ°еңЁй»ҳи®Өзҡ„ harnspec list дёӯ
  • д»ҚеҸҜжҗңзҙў
  • д»ҘеҗҺеҸҜд»ҘеҸ–ж¶ҲеҪ’жЎЈ

дҫқиө–е’Ңе…ізі»вҖӢ

еЈ°жҳҺдҫқиө–вҖӢ

зЎ¬дҫқиө–пјҲйҳ»еЎһпјүпјҡ

# еңЁSpecfrontmatterдёӯпјҲзӣ®еүҚеҝ…йЎ»жүӢеҠЁзј–иҫ‘пјү
depends_on:
  - spec-041  # еңЁжӯӨд№ӢеүҚеҝ…йЎ»е®ҢжҲҗ

жҹҘзңӢдҫқиө–вҖӢ

# жҳҫзӨәдҫқиө–е…ізі»еӣҫ
harnspec deps 042

иҫ“еҮәпјҡ

042-my-feature зҡ„дҫқиө–е…ізі»еӣҫ

дҫқиө–дәҺпјҡ
  вҶ’ 041-authentication [complete]

иў«йңҖиҰҒпјҡ
  вҶҗ 043-dashboard [in-progress]

зӣёе…іSpecпјҡ
  вҹ· 044-api-refactor [planned]

еҸҰи§Ғпјҡдҫқиө–

йӘҢиҜҒвҖӢ

жЈҖжҹҘSpecз»“жһ„е’Ңfrontmatterпјҡ

# йӘҢиҜҒжүҖжңүSpec
harnspec validate

# йӘҢиҜҒзү№е®ҡSpec
harnspec validate 042

жҚ•иҺ·пјҡ

  • ж— ж•Ҳзҡ„frontmatterз»“жһ„
  • зјәе°‘еҝ…йңҖеӯ—ж®ө
  • дҫқиө–еҫӘзҺҜ
  • Token и®Ўж•°иӯҰе‘ҠпјҲ>5,000 Token йҷҚдҪҺ AI жңүж•ҲжҖ§пјү

еҸҰи§ҒпјҡйӘҢиҜҒ

жҗңзҙўвҖӢ

йҖҡиҝҮеҶ…е®№гҖҒеҗҚз§°жҲ–е…ғж•°жҚ®жҹҘжүҫSpecпјҡ

# жҗңзҙўSpecеҶ…е®№
harnspec search "authentication"

# дҪҝз”ЁиҝҮж»ӨеҷЁжҗңзҙў
harnspec search "API" --status in-progress --tag backend

еҸҰи§ҒпјҡжҹҘжүҫSpec

еёёи§Ғе·ҘдҪңжөҒвҖӢ

ејҖе§Ӣж–°е·ҘдҪңвҖӢ

# 1. еҲӣе»әSpec
harnspec create auth-refactor --tags security,backend

# 2. зј–иҫ‘SpecпјҲе®ҡд№үй—®йўҳгҖҒи§ЈеҶіж–№жЎҲгҖҒжҲҗеҠҹж ҮеҮҶпјү
harnspec open 042

# 3. еҮҶеӨҮеҘҪж—¶з§»иҮіиҝӣиЎҢдёӯ
harnspec update 042 --status in-progress

е®ҢжҲҗе·ҘдҪңвҖӢ

# 1. ж Үи®°дёәе®ҢжҲҗ
harnspec update 042 --status complete

# 2. йӘҢиҜҒдёҖеҲҮд»Қз„¶жӯЈеёё
harnspec validate

# 3. еҸҜйҖүпјҡеҰӮжһңдёҚеҶҚйңҖиҰҒеҲҷеҪ’жЎЈ
harnspec update 042 --status archived

еӨҚжқӮSpecз®ЎзҗҶвҖӢ

# 1. жЈҖжҹҘSpecжҳҜеҗҰеҸҳеҫ—иҝҮеӨ§
harnspec view 042 --raw | wc -l

# 2. еҰӮжһң >400 иЎҢпјҢжӢҶеҲҶдёәеӯҗ Spec
# еҲӣе»ә DESIGN.mdгҖҒTESTING.md зӯү

# 3. йӘҢиҜҒз»“жһ„
harnspec files 042

жңҖдҪіе®һи·өвҖӢ

иҰҒеҒҡпјҡ

  • вң… дҪҝз”Ё harnspec update жӣҙж–°зҠ¶жҖҒ/дјҳе…Ҳзә§/ж ҮзӯҫпјҲж°ёиҝңдёҚиҰҒзј–иҫ‘frontmatterпјү
  • вң… дҝқжҢҒ README.md дё“жіЁпјҲдҪҝз”Ёеӯҗ SpecиҺ·еҸ–иҜҰз»ҶдҝЎжҒҜпјү
  • вң… йҡҸзқҖе·ҘдҪңиҝӣеұ•жӣҙж–°зҠ¶жҖҒ
  • вң… еҪ’жЎЈе·Іе®ҢжҲҗзҡ„Specд»ҘеҮҸе°‘ж··д№ұ
  • вң… е®ҡжңҹйӘҢиҜҒпјҲharnspec validateпјү

дёҚиҰҒпјҡ

  • вқҢ жүӢеҠЁзј–иҫ‘зі»з»ҹз®ЎзҗҶзҡ„frontmatterпјҲstatusгҖҒpriorityгҖҒtagsгҖҒassigneeгҖҒtransitionsгҖҒtimestampsпјү
  • вқҢ и®©SpecеңЁжІЎжңүжӢҶеҲҶзҡ„жғ…еҶөдёӢи¶…иҝҮ 400 иЎҢ
  • вқҢ и®©SpecдҝқжҢҒйҷҲж—§зҠ¶жҖҒ
  • вқҢ еңЁжҸҗдәӨеүҚи·іиҝҮйӘҢиҜҒ

дёӢдёҖжӯҘвҖӢ

  • жҹҘжүҫSpec - жҗңзҙўе’ҢеҸ‘зҺ°
  • Specз»“жһ„ - зҗҶи§ЈSpecз»“жһ„
  • йЎ№зӣ®з®ЎзҗҶ - зңӢжқҝи§Ҷеӣҫе’ҢеҒҘеә·и·ҹиёӘ
