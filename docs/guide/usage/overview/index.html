使用概览

HarnSpec 提供多种使用Spec的方式，具体取决于您的首选工作流程和用例。

快速开始​

如果您是 HarnSpec 新手，请从以下内容开始：

  1. AI 编码工作流 - 学习 AI 辅助开发的推荐工作流
  2. CLI 使用 - 掌握基本的 CLI 命令
  3. Spec结构 - 了解如何构建您的Spec

核心工作流​

AI 驱动的开发​

HarnSpec 专为 AI 优先的工作流程而设计：

  • AI 编码工作流 - 推荐的 SDD 工作流
  • Agent Skills - 通过 Agent Skills 使 AI 掌握方法论
  • 可视化模式 - 在 Web 界面中浏览Spec

命令行界面​

掌握 CLI 进行日常Spec管理：

  • 创建和管理Spec - 创建、更新和管理Spec
  • 查找Spec - 搜索和发现Spec
  • 项目管理 - 跟踪状态、依赖关系和健康状况

Spec结构​

了解如何编写有效的Spec：

  • Spec结构 - HarnSpec 的解剖结构

高级功能​

熟悉基础知识后：

  • 模板 - 自定义Spec模板
  • 自定义字段 - 扩展frontmatter
  • 变量 - 使用动态变量
  • frontmatter指南 - 高级frontmatter
  • 代理配置 - 配置 AI 代理

选择您的路径​

对于 AI 编码：

  1. 阅读 AI 编码工作流
  2. 设置 Agent Skills
  3. 使用 harnspec create 开始创建Spec

对于团队管理：

  1. 学习 CLI 使用
  2. 使用可视化模式获取概览
  3. 使用项目管理跟踪进度

对于高级用户：

  1. 掌握所有 CLI 命令
  2. 使用高级功能进行自定义
  3. 与您的工具链深度集成
